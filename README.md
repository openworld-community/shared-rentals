# shared-rentals

## Структура проекта

- sr-front - фронтенд приложение
- sr-back - бэкенд API
- configs - файлы конфигурации для nginx, базы
- docker - докерфайлы и docker-compose

## Стек проекта

### Frontend

- vite
- typescript
- react

### Backend

- nodejs
- [nestjs](https://nestjs.com/)
- [PostgreSQL](https://www.postgresql.org/)
